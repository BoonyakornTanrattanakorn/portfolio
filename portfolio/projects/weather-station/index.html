Overview

An end-to-end embedded IoT system collecting environmental data (temperature, humidity, pressure, UV index) and visualizing it on a live web dashboard. Built as a lab assignment but extended with a custom backend and a polished frontend.

System Architecture

STM32Sensor acquisition over I²C/SPI, forwards data to ESP32 via UART
ESP32-S3Wi-Fi bridge; publishes JSON payloads to MQTT broker (EMQX)
Go + FiberSubscribes to MQTT, stores readings in SQLite, serves REST & WebSocket
SvelteKitReal-time charts (Chart.js), historical graphs, responsive layout

Highlights

  • Sub-second end-to-end latency from sensor read to dashboard update via WebSocket push
  • Dockerised backend for reproducible deployment
  • Configurable alert thresholds with visual indicators on the dashboard

Overview

ThamRoi is a full-stack social application featuring group search, user profiles, and an admin moderation dashboard. I contributed both frontend and backend features across authentication, search, profile integration, admin reports, and deployment fixes.

Key Contributions

  • User consent UI during registration with enforced scroll-to-read behavior.
  • Group Search page UI and backend integration for live results and filtering.
  • Profile view wiring to backend APIs.
  • Admin reports system: JWT admin auth, report tagging, resolve/unresolve flows, and UI dialogs.
  • Backend: Prisma schema work, seed scripts, REST endpoints with query filters, and type-safe controllers.
  • Deployment fixes: env-based backend URL configuration.

Highlights

  • End-to-end features (UI + API) for search, profiles, and moderation.
  • Improved UX and legal compliance with ToS scroll requirement.
  • Data normalization and robust admin tooling for moderation workflows.

FPGA Real-Time Video Player

2025  ·  Academic Project  ·  Role: FPGA & RTL Developer
Verilog VGA FPGA SPI
No thumbnail yet

Overview

A fully hardware video playback pipeline on a Digilent Basys-3 (Artix-7) FPGA written entirely in Verilog. Raw 320×240 RGB565 frames are stored on a FAT-formatted SD card and streamed in real time to a VGA monitor at a target frame rate of 24 fps.

Architecture

  • SPI SD Controller — bit-bang / FIFO-backed module reading 512-byte blocks at 25 MHz
  • FAT16 parser — minimal state machine to locate and walk cluster chains
  • Dual-port BRAM ping-pong — one bank fills while the other feeds the VGA serializer
  • VGA timing generator — 640×480 @ 60 Hz (pixels 2× scaled, centred)

Key Challenge

Sustaining the SD read bandwidth above 2.4 MB/s while keeping the VGA pixel clock uninterrupted required careful pipeline staging and credit-based flow control between the FAT reader and the BRAM writer.
